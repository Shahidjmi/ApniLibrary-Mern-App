# Backend Setup
``` npm install```
This will start all the dependencies for the project
## Setup the Database
```MongoDB_URi = mongodb://localhost:27017/<db_name>
```PORT =<port_number>```
## START THE SERVER

```npm run start```

This will start the server and listen on the port specified in the .env file.
